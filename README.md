# Password Manager

A simple **Password Manager** built with **Python** and **Tkinter**. This application allows users to:

- Generate strong random passwords
- Copy passwords to the clipboard automatically
- Save website, email, and password entries in json format
- Easily manage and store credentials in a local file
- Has a search feature to search generated passwords

---

## 🛠 Features

- **Password Generator:** Generates random passwords with letters, numbers, and symbols.
- **Clipboard Integration:** Automatically copies generated passwords for convenience.
- **Save Credentials:** Stores website, email, and password combinations securely in a file (`data.txt`).
- **Simple GUI:** Built with `Tkinter` and `ttk` for a clean interface.
- **Search Feature** 

---

## 💻 Installation

1. Clone the repository:

```bash
git clone https://github.com/russel_yg/password-manager.git
cd password-manager
```
2. Install required Python packages:
```bash
pip install pyperclip
```
3. Run main.py
